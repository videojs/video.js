Video JS Skins

Replicas of some well known skins using just CSS

These skins were created to show how you can re-skin VideoJS using just CSS. To use these skins, download the CSS file and include in the same document as VideoJS. Then add the skin's classname to the video-js-box.

<div class="video-js-box vim-css">

The Original

No additional files needed.

VimCSS

Download VimCSS. Add the class "vim-css".

HuCSS

Download HuCSS. Add the class "hu-css".

TubeCSS

Download TubeCSS. Add the class "tube-css".
